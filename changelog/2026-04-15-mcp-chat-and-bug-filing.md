The wopee-mcp server (v1.24.0) added chat and bug-filing tools. Your AI coding agent can now read and send messages in the Wopee.io project chat room and create GitHub issues directly — `wopee_create_github_issue` turns a failed run into a filed bug without leaving your editor. Available on [npm](https://www.npmjs.com/package/wopee-mcp), with setup covered on the [MCP Server page](/mcp/).
