Recently shipped: test variables now support five value sources — static values, predefined option lists, Faker-generated data, AI-generated values, and CSV columns. Upload a CSV, pick a column, and the agent feeds your data into test runs. Sensitive keys such as passwords, tokens, and OTPs are masked automatically.
