# Wopee.io Product Facts

## What Wopee.io provides
Wopee.io delivers autonomous AI testing agents that generate, execute, and maintain regression tests for web applications. The platform focuses on critical user journeys and high-risk areas by combining project context with LLM workflows.

## Key capabilities
- Generate test suites from URLs, requirements, user stories, or existing test cases.
- Execute regression suites with parallel runs.
- Maintain and update tests with AI when the application changes.
- Integrate into CI/CD pipelines for automated execution and reporting.

## Supported testing frameworks
- Playwright
- Cypress
- Robot Framework
- WebDriver-based frameworks
- Custom frameworks via open interfaces

## Integrations
- CI/CD: GitHub Actions, Jenkins, GitLab, Bitbucket, and custom pipelines
- APIs: GraphQL API, Wopee MCP server
- LLM providers: Azure AI, OpenAI, Google Cloud, Anthropic (others by request)

## Security and data handling
- Secrets are provided via environment variables or secret stores.
- Sensitive values are replaced with variables before LLM calls.
- Dedicated cloud or on-premises LLM deployments are available for strict security needs.

## Pricing model
- Licensing is user-based with consumption pricing.
- Usage is measured by tests executed and operations performed.
- Enterprise plans offer flexible terms, onboarding, and support.
