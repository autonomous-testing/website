The Interactive Cockpit streams the agent's browser live while it tests. You can watch every action, nudge the agent with text, take over the browser yourself, and hand control back when you're done. When the agent is unsure, it asks — a multiple-choice question with a recommended option. And when it hits something only a human can do, such as a CAPTCHA or an MFA prompt, it requests a handoff instead of failing the run.
