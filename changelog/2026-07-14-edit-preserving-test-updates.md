title: "Your edits survive regeneration"
description: "Test-case regeneration in Wopee.io is now additive: an update mode reconciles new test cases with your manual edits instead of replacing them."
---

Regenerating test cases no longer overwrites your manual edits. Test-case generation now supports an update mode that extends and reconciles existing test cases instead of replacing them. A new "Keep existing test cases" toggle in the analysis tab puts you in control. Refine generated tests by hand, regenerate later, and keep both.
